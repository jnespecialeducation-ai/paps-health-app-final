# 커스텀 도메인 설정 가이드

## 현재 URL
```
https://paps-health-app.vercel.app/
```

## 목표
더 깔끔하고 기억하기 쉬운 URL로 변경
예: `https://paps-health.com` 또는 `https://paps-health.kr`

---

## 방법 1: Vercel에서 커스텀 도메인 추가 (권장)

### 1단계: 도메인 구매

도메인을 구매해야 합니다. 추천 사이트:

1. **가비아** (한국)
   - https://www.gabia.com
   - `.kr`, `.co.kr` 도메인 구매 가능
   - 예: `paps-health.co.kr`

2. **Namecheap** (해외)
   - https://www.namecheap.com
   - `.com`, `.net` 등 다양한 도메인
   - 예: `paps-health.com`

3. **GoDaddy** (해외)
   - https://www.godaddy.com
   - 다양한 도메인 확장자

### 2단계: Vercel에서 도메인 추가

1. **Vercel 대시보드 접속**
   - https://vercel.com
   - 프로젝트 `paps-health-app` 선택

2. **Settings → Domains**
   - 상단 메뉴에서 "Settings" 클릭
   - 왼쪽 사이드바에서 "Domains" 클릭

3. **도메인 추가**
   - "Add Domain" 또는 "Add" 버튼 클릭
   - 구매한 도메인 입력 (예: `paps-health.com`)
   - "Add" 클릭

4. **DNS 설정 안내 확인**
   - Vercel이 DNS 설정 방법을 안내합니다
   - 예시:
     ```
     Type: A
     Name: @
     Value: 76.76.21.21
     
     Type: CNAME
     Name: www
     Value: cname.vercel-dns.com
     ```

### 3단계: 도메인 등록 업체에서 DNS 설정

1. **도메인 등록 업체 대시보드 접속**
   - 가비아, Namecheap, GoDaddy 등

2. **DNS 관리 페이지로 이동**
   - "DNS 설정", "DNS 관리", "Domain Management" 등

3. **Vercel에서 제공한 DNS 레코드 추가**
   - Vercel이 안내한 A 레코드, CNAME 레코드 추가
   - 저장

4. **DNS 전파 대기**
   - 보통 5분~24시간 소요
   - 최대 48시간까지 걸릴 수 있음

### 4단계: 확인

1. **Vercel 대시보드에서 확인**
   - Settings → Domains
   - 도메인 상태가 "Valid Configuration"으로 변경되면 완료

2. **브라우저에서 접속 테스트**
   - `https://paps-health.com` (또는 설정한 도메인)
   - 웹사이트가 정상적으로 로드되는지 확인

---

## 방법 2: Vercel 기본 도메인 변경 (제한적)

Vercel의 기본 도메인은 프로젝트 이름에 따라 결정됩니다.

### 프로젝트 이름 변경

1. **Vercel 대시보드** → 프로젝트 선택
2. **Settings** → **General**
3. **Project Name** 변경
4. 새로운 기본 도메인 생성됨

⚠️ **주의**: 이 방법은 URL을 크게 개선하지 못할 수 있습니다.

## 비용

- **도메인 구매**: 연간 약 $10-15 (약 13,000-20,000원)
  - `.com`: 약 $10-15/년
  - `.co.kr`: 약 20,000원/년
- **Vercel**: 무료 (커스텀 도메인 추가 비용 없음)

## 빠른 시작 (가비아 예시)

1. **가비아 접속** → 도메인 검색 → 구매
2. **Vercel** → Settings → Domains → 도메인 추가
3. **가비아** → DNS 설정 → Vercel 안내에 따라 설정
4. **대기** → DNS 전파 완료 후 접속
